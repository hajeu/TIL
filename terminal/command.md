# 명령어

* ## pwd (print working directory)

    * ### pwd : 현재 위치의 경로를 나타낸다.

* ## ls (list segments)

    * ### ls : 현재 디렉토리에 존재하는 디렉터리와 파일 이름이 나타난다
    * ### ls -a : 모든파일(숨김파일 포함)을 출력한다.
    * ### ls -l : 파일의 세부사항을 출력한다.
    * ### ls -r : 파일 출력을 거꾸로 한다.

* ## cd (change directory)

    * ### cd : 디렉토리를 이동할때 사용한다.
    * ### cd .. : 상위 디렉토리로 이동한다.
    * ### cd ~ : 홈 디렉토리로 이동한다.

## 디렉토리 기호

### ./ : 현재 작업중인 디렉토리

### ../ : 현재 작업중인 디렉토리의 상위 디렉토리
